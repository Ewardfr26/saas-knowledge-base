# 🚀 SaaS Technical Support Knowledge Base & Troubleshooting Portfolio

Bienvenido a mi portafolio de Soporte Técnico enfocado en plataformas de software (SaaS) e infraestructura Cloud. Este repositorio funciona como una base de conocimientos (Knowledge Base) interna, donde documento simulacros de resolución de incidentes (L1/L2), análisis de logs, consultas a bases de datos relacionales y redacción de artículos técnicos para usuarios finales.

## 📁 Estructura del Repositorio

*   *   **`01-account-and-access/`**: Documentación para resolución de problemas de autenticación, gestión de accesos y credenciales.
    *   [Ver: Falla en inicio de sesión con Google (OAuth)](./01-account-and-access/error-inicio-sesion.md) 
*   **`02-data-and-performance/`**: Guías de diagnóstico para errores de carga de datos, lentitud del sistema e interacción con bases de datos.
    *   [Ver: Error al cargar archivos pesados (Módulo de Almacenamiento)](./02-data-and-performance/error-carga-archivos.md) 
*   **`03-integrations-and-apis/`**: Protocolos de soporte para fallas en webhooks, consumo de APIs y herramientas de terceros.

## 🛠️ Habilidades Evidenciadas en este Proyecto
*   **Troubleshooting Lógico:** Metodología clara de diagnóstico (Síntoma -> Causa -> Solución).
*   **SQL & Gestión de Datos:** Consultas relacionales simuladas para verificar estados de cuentas y logs de usuarios.
*   **Comunicación Técnica Empática:** Creación de plantillas de respuesta resolutivas y profesionales adaptadas al cliente (Macros).
*   **Documentación Técnica:** Estructura limpia utilizando Markdown para la gestión de conocimiento del equipo.
